# Backend - ML Components

This directory contains the machine learning backend for PayScope.

## Structure

- `models/` - ML model files and training scripts
- `data/` - Data processing and preprocessing scripts
- `api/` - Backend API endpoints for ML services
- `utils/` - Utility functions and helper scripts
- `config/` - Configuration files for ML models and services

## Getting Started

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set up your environment variables in `.env`

3. Run the API server:
```bash
python api/main.py
```

## Development

Add your ML models, data processing pipelines, and API endpoints in their respective directories.
